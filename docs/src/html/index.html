class model.CoSign1SModel(num_classes, stgcn_config_path=None, feat_dim=1024, lstm_hidden=512, dropout=0.2)[source]

Bases: Module

One-stream CoSign model with complementary masking.

Forward Pass Pipeline: 1. Spatial Extraction: Skeleton sequences pass through the ST-GCN backbone. 2. Branching & Masking: The feature map is duplicated. Branch 1 is multiplied by an attention mask (Phi),

and Branch 2 is multiplied by its inverse (1 - Phi).

  1. Temporal Compression: 1D Temporal CNNs (C3-P2) reduce sequence length.

  2. Contextualization: BiLSTM captures global sentence context.

  3. Classification: Shared Gloss Heads output final CTC log-probabilities.

Parameters:

  • stgcn_config_path (str)

PyTorch Dataset for loading continuous sign language sequences.
