workers

Worker threads module for managing parallel execution.

Separates heavy computer vision tasks (MediaPipe) and neural network inference from the main GUI thread to prevent application freezing.

class workers.AIWorker(shared_queue: Queue, mode: str = 'CSLR')[source]

Bases: QThread

Background worker thread for neural network inference.

Pops sliding windows from the queue, runs the PJM predictor, applies tracking and smoothing, and emits the final text to the UI.

Main execution loop for the AIWorker.

Constantly polls the queue for new sliding windows, predicts glosses using the neural network, filters results using tracking mechanisms, and pushes final text strings to the GUI and log file.
