\section{Opis problemów w trakcie realizacji projektu}

\subsection{Dataset}
\subsection{Text-to-PJM}
\subsection{PJM-to-text}

\begin{itemize}
    \item Niepoprawny design datasetu do CSLR
    \item Overfitting modelu CSLR na zdaniach, za mało próbek glossów w różnych kontekstach, żeby nauczyć model je rozróźniać
    \item Niejednoznaczny opis implementacji modelu cosign w artykule \cite{cosign}, co uniemożliwiło weryfikację implementacji
    \item Annotacje datasetów bywały sprzeczne, co zaniżało wyniki modelu
    \item Dataset składa się ze zdań, przez co nie jest adekwatny do nauki modelu ISLR, który wymaga pojedynczych słów
    \item Pretraining na datasetach LSFB, Pheonix-14T, nie przyniósł spodziewanych rezultatów; model nie miał lepszych wyników niż trening od zera. Możliwe, że wynika to z wątpliwej jakości nagrań i anotacji, których nie mamy jak zweryfikować
    \item Nikt z zespołu nie zna języka migowego, co utrudniało weryfikację wyników modelu i poprawki w annotacji datasetu
\end{itemize}
