    \item Overfitting modelu CSLR na zdaniach za mało próbek glossów w różnych kontekstach żeby nauczyć model je rozróźniać 
    \item Nie jednoznaczny opis implementacji modelu cosign w artykule \cite{cosign} co uniemożliwiło weryfikacje implementacji
    \item Annotacje datasetów bywały sprzeczne, co zaniżało wyniki modelu
    \item Dataset składa sie z zdań przez co nie jest adekwatny do nauki modelu ISLR, który wymaga pojedynczych słów
    \item Pretraining na datasetach LSFB, Pheonix-14T, nie przyniósł spodziewanych rezultatów, model nie miał lepszych wyników niż trening od 0 możliwe
        że wynika to z wątpliwej jakości nagrań i anotacji których nie mamy jak zweryfikować
    \item Nikt z zespołu nie zna języka migowego, co utrudniało weryfikację wyników modelu i poprawki w annotacji datasetu.

\end{itemize}}
